Программа web-page-stat

Автор:
Самохвалов Юрий Алексеевич (george.samyual@gmail.com)

Назначение: 
Программа предназначена для подсчёта частоты использования слов на web странице.
Вывод статистики осуществляется на консоль.

Запуск программы:
java -jar web-page-stat <URL страницы>

Примеры запуска:
java -jar web-page-stat https://yandex.ru
java -jar web-page-stat https://www.simbirsoft.com
